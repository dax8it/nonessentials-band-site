How to update the show list

This page exists so someone can update dates without touching GitHub, Netlify, code, or YAML. The Google Sheet is the source of truth.

Open the Google Sheet Back to the website

What to do

  1. Open the Google Sheet.
  2. Go to the Shows tab.
  3. Add one row per show.
  4. Keep the header row exactly as-is.
  5. Set status to live for any show that should appear on the site.

Required columns

  • status
  • starts_at
  • display_date
  • venue
  • address
  • notes
  • ticket_url

Important rules

  • Do not rename the column headers.
  • Do not delete the first row.
  • Use one show per row.
  • Use a real ISO date/time in starts_at.
  • Example: 2026-08-15T20:00:00-04:00

What happens automatically

  • The earliest upcoming show becomes the next-show card.
  • The countdown timer points to that same show.
  • Past shows disappear from the upcoming list.
  • Past shows move into the archive automatically.
  • If ticket_url is filled in, the site shows a Tickets / Info button.

Example row

status: live starts_at: 2026-08-15T20:00:00-04:00 display_date: Aug 15, Saturday, 8pm–11pm venue: Tommy Fox's address: 32 S. Washington Ave, Bergenfield, NJ notes: Classic rock all night. ticket_url: https://example.com/tickets

If something looks wrong

  • Check that status is live.
  • Check that starts_at is valid.
  • Check that nobody renamed a column header.
  • Refresh the website after saving the sheet.
